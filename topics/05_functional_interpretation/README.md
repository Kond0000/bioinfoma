# 05 Functional Interpretation

## この章の問い

差次的発現解析で遺伝子リストが得られたあと、それをどう研究上の意味に変えるのでしょうか。  
`IL6`、`MYC`、`TP53` という名前が並んでいるだけでは、細胞の中で何が起きたのかは十分に説明できません。

この章では、遺伝子リストを生物学的な解釈へつなげる考え方を学びます。

## なぜ機能解釈が必要か

RNA-seqの結果は、最初は数値と遺伝子名の集合です。しかし研究で知りたいのは、単に「どの遺伝子が上がったか」ではありません。より重要なのは、「それらの遺伝子が一緒に変化していることから、細胞でどのような反応が起きていると考えられるか」です。

たとえば、炎症関連遺伝子がまとまって上昇していれば、炎症応答が活性化している可能性があります。細胞周期関連遺伝子が変化していれば、増殖状態が変わっている可能性があります。

## この章で扱う考え方

- `gene set`
  共通する機能を持つ遺伝子の集まりです。
- `GO`
  Gene Ontology の略で、遺伝子の機能や生物学的プロセスを整理した体系です。
- `pathway`
  細胞内で連続して起こる反応やシグナル伝達のまとまりです。
- `enrichment`
  ある遺伝子リストに、特定の機能を持つ遺伝子が多く含まれているかを見る考え方です。

## 具体的に役立つ場面

差次的発現遺伝子の一覧を見て、炎症、増殖、ストレス応答などの生物学的テーマにまとめたい場面で役立ちます。

## この章で解き明かすこと

`IL6`、`MYC`、`TP53` の変化から、刺激された細胞で何が起きていそうかを説明できるようにします。

## 読み取れるようになること

遺伝子名の羅列を、研究上の文章に変換できるようになります。最終的には、「刺激群では炎症応答と細胞ストレス応答の活性化が示唆される」のように、解析結果を生物学的な主張としてまとめる力を身につけます。
