# 05 Functional Interpretation

## このトピックでやること

差次的発現遺伝子の機能を、GOや経路解析で解釈します。

## 目的

- 遺伝子リストから生物学的意味を引き出す
- 単なる遺伝子名の列で終わらせない
- 論文の結果節に近い考え方を身につける

## 具体的に役立つ場面

差次的発現遺伝子の一覧を見て、「炎症なのか、増殖なのか、ストレス応答なのか」といった生物学的テーマにまとめたい場面で役立ちます。

## このトピックで解く課題

「IL6 や MYC や TP53 が変化したとき、この実験では細胞で何が起きていそうか」を、機能のまとまりとして説明できるようにします。

## 教材の内容

- 遺伝子リストだけでは解釈が足りない理由
- GOや経路解析が何を助けるのか
- 生物学的な要約文の組み立て方

## ゴール

発現変動の結果を「どんな生物学的変化が起きたか」に結びつけられることです。
