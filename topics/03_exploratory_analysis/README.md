# 03 Exploratory Analysis

## この章の問い

正規化したあと、すぐに差次的発現解析へ進んでよいのでしょうか。  
その前に、データ全体の形を見て、刺激群と対照群が本当に違って見えるか、変なサンプルが混ざっていないかを確認する必要があります。

この章では、PCAとヒートマップを使って、RNA-seqデータ全体の構造を読みます。

## なぜ探索的解析が必要か

RNA-seqでは、1つの遺伝子だけを見ると変化しているように見えても、データ全体としては条件差が弱いことがあります。逆に、1つのサンプルだけが極端に外れていて、その影響で差が大きく見えることもあります。

探索的解析は、統計検定に入る前の地図読みです。データの全体像を確認し、解析を進めてよいかを判断します。

## この章で扱う考え方

- `PCA`
  多数の遺伝子の発現パターンを、少数の軸に要約してサンプル間の近さを見る方法です。
- `ヒートマップ`
  遺伝子ごとの発現量を色で表し、どのサンプルでどの遺伝子が高いかを見る図です。
- `外れ値`
  他のサンプルと比べて極端に違うサンプルです。実験ミス、処理条件の違い、バッチ効果などが原因になることがあります。

## 具体的に役立つ場面

差次的発現解析に進む前に、炎症刺激群と対照群が全体として分かれているか、1サンプルだけ変ではないかを確認する場面で役立ちます。

## この章で解き明かすこと

このデータは刺激応答を持っていそうか、それとも外れ値やノイズが強そうかを、PCA とヒートマップで見極めます。

## 読み取れるようになること

PCAでサンプル同士の距離を読み、ヒートマップで遺伝子ごとの発現パターンを読む力を身につけます。解析結果を見たときに、「このデータは先に進めてよさそうか」を判断できるようになることが目標です。
