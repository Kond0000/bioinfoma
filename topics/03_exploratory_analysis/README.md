# 03 Exploratory Analysis

## このトピックでやること

PCAやヒートマップで、サンプル同士の近さや外れ値の有無を確認します。

## 目的

- 条件ごとにサンプルがまとまるかを見る
- 外れ値やバッチ効果に気づけるようにする
- 可視化の基本をGitHubに残せる形にする

## 具体的に役立つ場面

差次的発現解析に進む前に、「treated と control は全体として別れているのか」「1サンプルだけ変ではないか」を確認する場面で役立ちます。

## このトピックで解く課題

「このデータは条件差をちゃんと持っていそうか、それとも外れ値やノイズが強そうか」を、PCA とヒートマップで見極めます。

## 教材の内容

- PCAでサンプル間の近さを見る考え方
- ヒートマップで発現パターンを読む視点
- 外れ値やバッチ効果を疑う場面

## ゴール

「このデータは解析を先に進めてよいか」を可視化から判断できることです。
