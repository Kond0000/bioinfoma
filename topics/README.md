# Topics Overview

このディレクトリは、RNA-seq学習をトピック単位で整理したものです。

## 読み方

1. 番号の若いトピックから順に進める
2. 各 `README.md` で目的とゴールを確認する
3. `scripts/` を実行して結果を確かめる
4. `results/` の図や表を見て理解を言語化する

## トピック一覧

### 01 Count Matrix QC

入口です。count matrix を読み込み、サンプルごとの総カウント数を確認します。  
詳細: [01_count_matrix_qc/README.md](01_count_matrix_qc/README.md)

### 02 Normalization

### 03 Exploratory Analysis

### 04 Differential Expression

条件間で変化した遺伝子を調べます。  
詳細: [04_differential_expression/README.md](04_differential_expression/README.md)

### 05 Functional Interpretation
