## このトピックでやること

これまでの解析結果を、GitHubで見やすい形に整理します。

## 目的

- READMEで解析目的と結果を伝える
- 図とコードの対応が分かる構成にする
- 研究職応募でも見せやすい形に整える

## 具体的に役立つ場面

自分の解析を指導教員、共同研究者、採用担当者に見せるときに、「何を理解していて、どこまで自力で進められるか」を短時間で伝える場面で役立ちます。

## このトピックで解く課題

「このRNA-seqケーススタディで何が分かり、なぜその結論に至ったのか」を、README と図だけで他人に伝わる形にまとめます。

## 教材の内容

- 図と説明の対応づけ
- GitHub上で読みやすいREADMEの組み立て

## ゴール

他人が見ても「何を、どう解析し、何が分かったか」が短時間で伝わることです。
