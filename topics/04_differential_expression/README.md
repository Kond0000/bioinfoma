# 04 Differential Expression

## この章の問い

炎症刺激によって、本当に変化した遺伝子はどれでしょうか。  
単に平均値が違うだけではなく、その差がどの程度大きく、どの程度信頼できるのかを同時に見る必要があります。

この章では、差次的発現解析の結果表と volcano plot の読み方を学びます。

## 差次的発現解析とは

差次的発現解析は、条件間で発現量が変化した遺伝子を見つける解析です。RNA-seqでは通常、数千から数万の遺伝子を同時に調べます。そのため、ただ平均値を比べるだけではなく、ばらつきや多重検定の影響を考える必要があります。

この教材では、まず結果表の読み方に集中します。実務では DESeq2 や edgeR などの専用ツールを使うことが多いですが、それらの出力を読むためにも、基本概念を先に理解しておく必要があります。

## この章で扱う考え方

- `fold change`
  条件間で発現量が何倍変わったかを表します。
- `log2FoldChange`
  fold change を log2 スケールにした値です。正なら刺激群で高く、負なら刺激群で低いことを意味します。
- `p-value`
  観測された差が偶然だけで起こる可能性を表す指標です。
- `adjusted p-value`
  多数の遺伝子を同時に検定するために補正した p-value です。
- `volcano plot`
  横軸に変化量、縦軸に統計的な強さを置いた図です。

## 具体的に役立つ場面

炎症刺激の前後で、どの遺伝子が本当に変化したのかを候補として絞り込みたい場面で役立ちます。

## この章で解き明かすこと

刺激によって本当に変化した遺伝子はどれかを、fold change と有意性の両方を見ながら判断します。

## 読み取れるようになること

差次的発現解析の表を見て、単に「p-value が小さい遺伝子」ではなく、「変化量が大きく、統計的にも信頼できる遺伝子」を選べるようになります。
